# Frontend Assignment

This is a simple React frontend for the assignment.

Stack:
- React 18
- Vite dev server

Features:
- Login and register pages
- Dashboard page with notes list
- Create, update and delete notes
- Uses JWT token stored in localStorage
- Shows success and error messages

How to run:
1. Go to the frontend folder.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the shown localhost url (default http://localhost:5173).

Make sure the backend is running on http://localhost:8080.
